# Agregador de Dados para Imposto de Renda (Excel)

Projeto desenvolvido em Excel com o objetivo de organizar e centralizar informações essenciais para a declaração do imposto de renda.
A ferramenta permite o controle de dados por meio de planilhas organizadas, validações automáticas, fórmulas e uma interface simples e intuitiva.

## Funcionalidades
- Organização de dados financeiros
- Validação automática de campos
- Uso de fórmulas no Excel
- Interface amigável
